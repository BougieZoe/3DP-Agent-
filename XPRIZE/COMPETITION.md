# COMPETITION.md — XPRIZE Strategy

## Target Category
**Small Business Services** — Powering everyday 3D printing businesses with tools to compete and win.

## Judging Criteria (Equal Weight)
1. **Business Viability**: Real revenue + sustainable model + user evidence
2. **AI-Native Operations**: AI agents running key business functions in production
3. **Category Impact**: Helping small 3D print businesses save time/money/material

## Key Evidence Needed
- Stripe revenue + monthly breakdown
- Firebase usage logs + real user testimonials
- Antigravity/Gemini agent execution logs
- P&L using official template

## Strategy
- Freemium model (free limited analyses → Pro subscription)
- Target: 3D print studios, prototype designers, small manufacturers
- Emphasize multi-agent debate + causality visualization as unique AI-Native strength
