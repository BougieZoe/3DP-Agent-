# 3DP Agent

AI-powered 3D printing analysis tool. Upload STL files for geometry analysis, printability assessment, and AI consultation.

## Features
- STL geometry analysis pipeline
- Printability heuristics
- Multi-language support (EN/JA/ZH)
- AI chat with Claude / GPT / Gemini / DeepSeek

## Tech Stack
TypeScript · React · Three.js · Vite
